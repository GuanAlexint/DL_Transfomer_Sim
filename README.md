# 基于 Transformer 与 SimCSE 的患者主诉多症状智能识别

一个端到端的深度学习系统，用于从非结构化的患者主诉文本中自动识别9种核心抑郁相关症状。

## 📌 核心创新
- **自监督预训练**：采用 SimCSE 框架，利用 dropout 增强学习更鲁棒的中文医疗句子表示。
- **多模态融合**：结合文本信息（Transformer Encoder）与结构化特征（年龄、性别、病史）。
- **全流程实现**：从零实现了数据加载、词表构建、模型训练、评估到部署准备的全部环节。

## 📊 性能
- **Micro-F1**: 0.7019
- **Macro-F1**: 0.5183
- **Mean AUC**: 0.8912

## 🛠 技术栈
- Python, PyTorch, Jieba, Scikit-learn

## 💡 亮点
- **从零造轮子**：手写 Transformer 和 SimCSE，底层++。
- **解决真实痛点**：为电子病历结构化提供自动化工具，提升临床效率。
- **前沿技术**：成功将 SimCSE 应用于小样本医疗 NLP 场景。
